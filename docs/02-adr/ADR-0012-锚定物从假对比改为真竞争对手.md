# ADR-0012: 锚定物从假对比改为真竞争对手

**日期:** 2026-04-23
**状态:** Accepted
**决策者:** Bob Qiushao
**层级:** Commercial
**关联 ADR:** ADR-0010 (上游)

---

## 背景（Context）

原 ROI 计算器使用"自建团队 ¥79 万"作为对比锚点。
这个数字无法被买家验证，容易被质疑为虚假对比，反而降低可信度。
需要切换为买家可以自行核实的真实市场对手。

## 决策（Decision）

销售文案中的对比锚改为对比真实市场竞争对手：

| 对手类型 | 价格区间 | Bob 的差异化 |
|---|---|---|
| 瑞诺/三行外贸建站套餐 | ¥30,000-80,000/年 | 定制化 + 能力组合 + 持续运维 |
| Shopify + 插件 + 开发 | ¥25,000-40,000/年 | 数据自有 + 零平台抽成 + 定制前端 |
| 普通接单个人 | ¥3,000-8,000 | 定制（非模板）+ 实战验证 + 持续支持 |

## 备选方案（Alternatives Considered）

| 方案 | 优点 | 缺点 | 为什么没选 |
|---|---|---|---|
| 维持"自建团队"对比 | 数字大，视觉冲击强 | 不可验证，被质疑时崩溃 | 可信度风险 |
| 不做对比 | 无争议 | 失去价格锚定效果 | 放弃销售杠杆 |
| 真实对手对比（本方案） | 可验证、可信、有说服力 | 需要持续更新市场数据 | 已接受 |

## 后果（Consequences）

### 正面
- 买家可自行核实，大幅提升可信度
- 明确定位 Bob 在市场中的位置（高于个人接单，低于大型外包）
- 差异化说明更清晰（定制 vs 模板）

### 负面 / 已知风险
- 竞争对手价格会变化，需要定期更新
- 部分买家可能选择更便宜的竞争对手

### 触发重新评估的条件
- 每半年更新一次竞争对手价格数据
- 当市场格局发生重大变化时

## 实施清单（Implementation）

- [x] ROISection 更新锚定物为真实竞争对手
- [x] 三语言同步更新
- [ ] 建立竞争对手价格定期追踪机制
