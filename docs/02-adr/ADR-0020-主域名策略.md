# ADR-0020: 主域名策略

**日期:** 2026-04-23
**状态:** Accepted
**决策者:** Bob Qiushao
**层级:** Technical
**关联 ADR:** ADR-0021 (下游)

---

## 背景（Context）

个人品牌需要一个稳定的主域名作为所有子域名的根。
Bob 已有 `bobqiushao.online`，并在此基础上积累了一定的 SEO 权重和用户认知。
需要明确主域名策略，避免未来因域名变更造成资产损失。

## 决策（Decision）

- 主域名继续使用 **bobqiushao.online**（已有积累，不换）
- 子域名按客群 / 案例 / 能力命名（详见 ADR-0021）
- 客户域名由客户自行购买，Bob 协助挂载

**子域名命名规范：**

| 用途 | 格式 | 示例 |
|---|---|---|
| 客群落地页 | `{segment}.bobqiushao.online` | `ecom.bobqiushao.online` |
| 案例展示 | `case-{name}.bobqiushao.online` | `case-auraloop.bobqiushao.online` |
| 工具/服务 | `{tool}.bobqiushao.online` | `sync.bobqiushao.online` |

## 备选方案（Alternatives Considered）

| 方案 | 优点 | 缺点 | 为什么没选 |
|---|---|---|---|
| 换用短域名（如 bob.work） | 更简洁 | 已有积累归零，成本高 | 时机未到 |
| 多个独立域名 | 各自独立 | 管理成本高，SEO 分散 | 不符合单人运营 |
| 维持 bobqiushao.online（本方案） | 保留积累，低成本扩展 | 域名稍长 | 已接受 |

## 后果（Consequences）

### 正面
- 保留已有 SEO 积累和用户认知
- 子域名扩展成本极低（¥80/年主域名 + Vercel 免费版）
- 统一品牌形象

### 负面 / 已知风险
- `bobqiushao.online` 域名较长，不易记忆
- 未来品牌升级时可能需要迁移

### 触发重新评估的条件
- 当个人品牌发展到需要更短域名时
- 当 `bobqiushao.online` 续费成本大幅上升时
