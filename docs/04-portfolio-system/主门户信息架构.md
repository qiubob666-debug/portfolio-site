## 五、域名前缀选项库(给客户参考)

按 ADR-0021,提供前缀选项库:

| 用途 | 推荐前缀 |
|---|---|
| 主商城 | shop / store / mall |
| 子品牌 | 品牌名 / brand-{name} |
| 内容博客 | blog / journal / insights / stories |
| 帮助中心 | help / support / docs / faq |
| 测试 / 活动页 | try / new / lp / launch |
| 多语言 | en / de / jp / fr / es |
| 区域 | us / eu / asia / cn |
| 客户专属 | members / vip / pro |
| API / 后端 | api / app / admin |

**对外销售话术:**
> 你不需要为每个站点买新域名。一个域名 + 多个前缀,Vercel 挂载,30 秒上线。
> 一年成本:一个域名(几十块)+ Vercel(免费版即可)= 几乎为零。
> 这比 Shopify 一个店铺月费 $29 一年 ¥2,500 都要便宜。
